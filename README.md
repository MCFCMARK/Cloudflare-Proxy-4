# Cloudflare CORS Proxy

Simple streaming CORS proxy using Cloudflare Workers.

## Usage

```txt
https://your-worker.your-subdomain.workers.dev/https://example.com/file.mp4
```

## Deploy

Click the button below:

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME)
